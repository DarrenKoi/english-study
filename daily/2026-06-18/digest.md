# 오늘의 표현 — 2026-06-18

설계문서가 결정을 *정당화하는* 영어 — 대조·결과·경고를 잇는 표현들.

1. **If the wiki adds work for the user, it has failed.** — 목표를 *반대 상황으로* 못 박는 단정형. "X 하면 그 설계는 실패다."
2. **degrade gracefully** — 의존 서비스가 죽어도 멈추지 않고 *품질만 낮춰* 계속 동작하다.
3. **made structurally impossible / safe by construction** — 검사로 막는 게 아니라 *구조 자체가* 위험을 불가능하게.
4. **surfaced, never silently reconciled** — 충돌은 숨겨 합치지 말고 *드러내라*.
5. **masquerade as** — 어떤 실패가 다른 정상 상태인 *척 위장하다* (그래서 더 위험).
6. **shadow mode / kill switch** — 결정 반영 없이 점수만 기록하며 검증 / 문제 시 즉시 끄는 비상 플래그.
7. **the system's own case law** — 자기 실수에서 쌓은 누적 규칙을 *판례*에 빗댄 비유.
