# spool/ — 학습 노트 드롭 폴더

자기 전이나 생각날 때, **궁금한 것·공부하고 싶은 것을 이 폴더에 그냥 파일로 적어두세요.**

- 형식: `.md` 또는 `.txt`. 파일명은 자유 (예: `affect-vs-effect.txt`, `poetic.md`).
- 내용: 자유 질문. 예) "affect 와 effect 차이", "'resilient' 들어간 예문 10개",
  "왜 a historic 이라고 하나?", "이 문장 더 자연스럽게 고쳐줘: ...".

## 언제 처리되나
- **야간 무인**: 매일 23:00 launchd 실행이 자동으로 답합니다.
- **즉시**: Claude Code 에서 `/study` 를 실행하면 그 자리에서 답합니다.

spool 노트는 **최우선**으로 처리됩니다(프로젝트 docs·대화보다 먼저).

## 처리 후
- 답변: `daily/<날짜>/spool/<노트이름>.md` (맨 위에 원 질문 인용).
- 원본 노트: 이 폴더에서 비워지고 `spool/done/<날짜>-<노트이름>` 으로 **아카이브**됩니다
  (지워지지 않으니 나중에 다시 찾을 수 있습니다).
